# Changelog

## v1.0.0

### Fitur Utama
- Buat invoice dengan 3 template (Minimalis, Profesional, Kreatif)
- Download invoice sebagai PDF
- Kelola data klien
- Dashboard dengan statistik bisnis
- Multi-currency (IDR, USD, EUR, SGD, MYR, JPY, GBP)
- Mode gelap dan terang

### Fitur Lanjutan
- Invoice berulang (bulanan, triwulan, tahunan)
- Reminder pembayaran
- Impor data dari CSV
- Ekspor invoice ke Excel
- Backup dan pulihkan data lokal
- Sinkronisasi antar perangkat (via cloud)

### Keamanan
- Login dengan email dan password
- Login dengan kode OTP
- Autentikasi admin terpisah
